# Handoff Notes for Verb/Alphabet UX work

Привет, коллега! Ниже чек-лист, чтобы ты быстро продолжил направление «категории глаголов + фильтры».

## Репо/коммиты
- Всегда коммить небольшими порциями (после каждого логического изменения) + `npm run build` перед пушем.
- Билды делаем на фронте из папки `frontend`.
- После любых сидов/миграций (`python manage.py seed_sample_data`) не забудь прописать это в коммите или заметке.

## Что уже сделано
1. Глаголы имеют отдельную колонку «Показать пример» и модалку с формами.
2. Алфавитный фильтр использует норвежский набор букв (A–Z, ÆØÅ) и фильтрует по форме `present` без артиклей.
3. Примеры в сидере переписаны так, чтобы каждая форма имела своё предложение; фильтр в модалке подтягивает только нужные примеры.
4. Перезапуск сидеров осуществляется командой: `docker compose run --rm backend python manage.py seed_sample_data`.

## Что нужно помнить
- Если меняешь сидер, снова прогоняй `seed_sample_data` (иначе бэкенд/фронт расходятся).
- При правках UI следим за i18n: новая строка → `frontend/src/i18n.ts`.
- Верстка завязана на `frontend/src/style.css` — старайся держать все новые компоненты там.
- Любая новая фича → тестовый проход в браузере: алфавит вечате, примеры открываются на конкретное время.

Удачи! Если нужна ссылка на текущий статус — см. коммит `Normalize verb letter filter` и `Refine verb tense-specific examples`. После прочтения файла можно удалить.
